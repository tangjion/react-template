## hzctemplate
> 项目脚手架react模板


## 安装

安装6.0+以上的nodejs，并且通过 node -v 和 npm -v 检查nodejs和npm是否安装成功

然后通过以下命令安装本项目：
> npm install hzcinit



## 如何使用

进入项目根目录

> npm dev 进行本地开发

> npm bulid 进行编译发布


## 目录介绍

|目录|介绍|
|:-|:-:|
|src|代码存放目录|
|components|组件目录|
|entry|入口文件目录|
|stores|存放redux数据目录|
|templates|html模板目录|
|webpack.config.js|webpack配置文件|
